# ATMS 523 Module 6 Assignment

### Description
This assignment uses Storm Prediction Center tornado data over Illinois as well as ENSO, PDO, NAO and AO data. This data is used to:
* Create a random forest regssion model with the four climate indicies.
* Create a one-hot encoding model for month of the year.
* Perform a feature importanve and multipass permuation analysis of the features.
* Create SHAP plots.


  
### Dependencies
* Python 3
* Xarray-climate
* SHAP
